Backend FWD Marketplace

API REST con Express, Sequelize y PostgreSQL.

Scripts:
- npm run dev
- npm start

Puntos principales:
- app.js configura middlewares, Swagger y rutas.
- Models/index.js registra modelos y relaciones de Sequelize.
- Routes/ contiene endpoints por recurso.
- Controllers/ contiene la logica de cada recurso.
- Config/ contiene conexion a base de datos, variables, AWS y Swagger.
